    2. Agent专用搜索 AnySearch 登顶 Product Hunt，Token 更省搜得更准

    来源: 量子位 | 作者: 闻乐 | 关键词: Agent搜索, AnySearch, ProductHunt, Token优化

    要解决的问题

    现有 AI 搜索把网页链接和摘要丢给 Agent，导致 Agent 需要读取大量广告和 SEO 垃圾信息的全文，上下文冗余严重、Token 消耗大、信息准确度低，尤其在涉及垂直领域数据时搜不到有效信息。

    已有做法

    Perplexity、Exa、Tavily 等搜索工具面向人类设计，或多或少的把一堆结果全部返回给 Agent 让模型自己烧 Token 筛选，缺乏对 Agent 工作流的针对性优化，且海外搜索引擎对国内本土信息覆盖不足。

    本文做法

    中国团队推出的 AnySearch 为 Agent 重新设计了搜索链路： 1. 智能意图识别：自动选择最合适的数据路径（代码走 GitHub、金融走实时数据库、法律走司法文书） 2. 20+垂直领域数据源：覆盖代码、法律、学术、金融、安全、企业商业等，直接连接网页之外的结构化数据 3. 多源交叉过滤+去重排序：降低同站重复内容权重，信息丰富度优先，语义+时效双排序 4. 内容清洗转 Markdown：自动提取正文、去噪、结构化，直接交付可推理数据 5. Agent原生设计：支持 API/MCP/Skill 接入，自动容错和超时管控

    最终效果

    • 综合准确率 76.4%，延迟优于 Parallel 和 Brave Search
    • 对国内本土信息覆盖显著优于 Exa（企业合规记录等海外搜索完全缺失）
    • 用户实测：替换后量化交易 Agent 判断失误减少，系统更稳定
    • Product Hunt 周榜 Top 1
