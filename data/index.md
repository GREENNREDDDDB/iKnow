# 欢迎来到 iKnow 个人资讯库

这是一个由 **Hermes Agent** 驱动的全自动个人资讯收集与整理系统。

## 功能介绍

- **多源汇聚**：通过 Agent 定时抓取指定目标网站（如 HackerNews、机器之心等）的核心内容。
- **自动总结**：大模型对内容进行去重、深度总结，并自动生成结构化 Markdown 文档。
- **关系图谱**：提取跨文章的实体与关系，生成可视化的知识图谱。
- **多端同步**：文件实时同步至 GitHub，通过 GitHub Pages 提供全平台网页访问（如微信机器人推送网页链接）。

## 导航说明

请使用顶部或侧边栏导航：
- **文档浏览**：查看所有自动生成的每日资讯简报与总结报告。
- **关系图谱**：查看可视化的知识关系网络。
